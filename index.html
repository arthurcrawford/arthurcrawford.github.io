Simplest .deb

Tue, Jan 26, 2016

We are to build a package called margherita. Let’s examine the source code tree.

# tree margherita
margherita
|-- DEBIAN
|   `-- control
`-- usr
    `-- bin
        `-- mozzarella

The source contains our “application”, simply an executable shell script called mozzarella. Note that the relative path of the executable in the source tree reflects its intended destination when installed - i.e. /usr/bin.

To build this package we can use the tool dpkg.

# dpkg --build margherita build

You can do the same thing this way:

# dpkg-deb --build margherita build

A quick look at the manpage for dpkg will reveal that it simply delegates some of its capabilities to lower level tools - hence the alternative form.

In either case, the --build option takes a single argument - the source directory (margherita/). The second positional argument is the target directory into which the archive is built (build/).

You may be interested to see what’s actually in the built package. A Debian package is an ar format archive. You can look inside it using the ar tool.

# ar t build/margherita_1.0.0_all.deb
debian-binary
control.tar.gz
data.tar.xz
